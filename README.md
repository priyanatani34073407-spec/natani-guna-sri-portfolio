🌐 Personal Portfolio Website

📌 Project Overview

This is a responsive Personal Portfolio Website developed using HTML5. It showcases personal information, educational background, technical skills, projects, resume upload form, contact details, social media links, multimedia content, and college location.

The website serves as an online portfolio to present academic achievements, projects, and career objectives.

---

✨ Features

- 🏠 Home section with introduction
- 👩 About Me section
- 🎓 Education details in a table
- 💻 Technical Skills section
- 🚀 Projects with GitHub links
- 📄 Resume Upload Form
- 📬 Contact Form
- 📧 Email and Phone contact buttons
- 🔗 LinkedIn and GitHub profile links
- 🎵 Self Introduction Audio
- 🎥 Introduction Video
- 📍 Google Maps College Location
- 📱 Clean and responsive layout

---

🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Optional for future enhancements)
